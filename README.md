## 🚀 Overview

This repository contains my personal Emacs configuration file (`init.el`), used to set up the environment, load necessary packages, and configure custom keybindings and settings.

---

## 🏁 Quickstart

1. Back up your existing configuration:
   ```bash
   mv ~/.emacs.d/init.el ~/.emacs.d/init.el.backup
   ```
2. Copy the new configuration:
   ```bash
   cp init.el ~/.emacs.d/init.el
   ```
